Memory Game

Welcome to the Memory Game, a fun and interactive way to test your recall skills! Built with React.js, this game challenges you to match pairs of cards. Each turn, you can flip two cards to reveal the images. The goal is to match all the pairs in the fewest turns possible. The game shuffles cards randomly each time you start a new session, ensuring a unique experience every round.

Digital Watch

This React-based digital clock allows users to view real-time updates of the current time in various countries, including Pakistan. The clock dynamically fetches the local time of a selected region using the IPGeolocation API. Users can interact with the app by choosing their preferred location from a dropdown menu, with the time displayed in a clean, responsive format that updates every second. The design also features a sleek flip clock effect for an enhanced visual experience.
